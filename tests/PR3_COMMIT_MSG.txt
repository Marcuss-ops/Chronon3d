test(pr3): 4 end-to-end compositions with golden PNG pairs

PR3 completes the composition-coverage layer started in PR2:

  1. text+gradient+mask        (320x240) - l.mask_rect on title layer
  2. camera+depth+DoF          (480x270) - Camera2_5D::dof enabled
  3. motion blur+transp.       (320x240) - TemporalAccumulation 16s 360deg
  4. video+images+RTL text     (480x270) - Arabic RTL via NotoNaskhArabic
                                    + image proxy (no .mp4 fixture shipped)

Each composition renders as a none-vs-effect-on golden PNG pair (8 PNGs
total in tests/golden/pr3/), with explicit non-equivalence assertions on
fb_hash between the two variants so a regression that silently disables
an effect cannot pass the golden check.

Reused PR2 infrastructure:
- tests/helpers/test_utils.hpp (ctt::make_renderer / framebuffer_hash /
  average_luma_rect)
- tests/visual/support/golden_test.hpp (verify_golden + ImageDiffThreshold
  max_rmse=0.012, max_changed=0.08)
- asset path conventions from test_main.cpp mounts

New executable: chronon3d_pr3_composition_visual_tests, registered in
CHRONON3D_RENDER_TEST_DEPS.
